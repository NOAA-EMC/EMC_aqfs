UPCASE()

Fortran version: 

    SUBROUTINE UPCASE( ASTRING )
        CHARACTER*(*)  ASTRING    ! string to make into all caps

NO C version:

Summary:

Upcases (turns into ALL CAPS) the contents of the Fortran character string ASTRING.

Preconditions:

For Fortran-90 declarations and interface checking: 
    USE M3UTILIO

Fortran Usage: 

    ...
    CHARACTER*256  ASTRING
    ...
    ASTRING = 'sOmeThiNg WiERd'
    ...
    CALL UPCASE( ASTRING )   !  now ASTRING is all-caps: 'SOMETHING WIERD'
    ...

C Usage:

don't
